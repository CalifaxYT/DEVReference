# Encapsulation in C++
## A quick reminder about adding encapsulation in C++

Strings are **ALWAYS** enclosed in two DOUBLE quotes:
```cpp
cout << "Hello, World" << endl;
```

Chars are **ALWAYS** enclosed in two **SINGLE** quotes:
```cpp
cout << 'a' << endl;
```
